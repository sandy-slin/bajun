# A股基本信息获取系统

## 项目概述
开发一个Python程序，用于获取和分析A股市场的基本信息和交易数据。

## 功能需求

### 核心功能
1. **历史交易数据获取**
   - 获取过去半年的A股交易数据
   - 实现本地缓存机制（优先本地读取，获取后缓存）

2. **智能分析功能**
   - 使用DeepSeek LLM分析过去一个月和未来一个月的重大事件信息
   - 分析相关版本或概念过去一周的动向
   - 生成未来三天的交易建议
   - 分析结果采用 Markdown 格式，并缓存到本地

### 技术要求
- 使用Python 3
- 在venv虚拟环境中运行
- 遵循`.claude/CLAUDE.md`的架构要求
- 依赖管理通过`requirements.txt`
